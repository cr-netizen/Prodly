# Prodly

Prodly is an AI-powered Food Processing Product Description Generator for e-commerce listings.

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas (Week 5)

---

## How to Run Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

---

## How to Run Backend

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```
http://localhost:5000
```

---

## API Endpoints

GET /api/products

GET /api/products/:id

POST /api/products

PUT /api/products/:id

DELETE /api/products/:id

GET /api/products/search?q=
